# Contributing to Cascavel Secret Scanner

Thank you for your interest in contributing to Cascavel Secret Scanner!

## Adding New Patterns

1. Edit `scanner.sh`
2. Add your pattern to the `PATTERNS` array following the format:
   ```
   "pattern-id|severity|Description|REGEX|CWE-XXX"
   ```
3. Test your pattern against known samples
4. Submit a pull request

## Guidelines

- Follow the existing code style
- Test your changes locally before submitting
- Include examples of what your pattern detects
- Reference the relevant CWE identifier

## Code of Conduct

Be professional, respectful, and constructive.
