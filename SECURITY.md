# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in Cascavel Secret Scanner, please report it responsibly:

1. **Email**: contato@rettecnologia.org
2. **GitHub Security Advisory**: [Report here](../../security/advisories/new)

**Do NOT open a public issue for security vulnerabilities.**

We will acknowledge your report within 48 hours and aim to provide a fix within 7 days for critical issues.

## Security Considerations

- This tool scans for known secret patterns and may produce false positives
- Ensure your workflow logs do not expose sensitive information
- The scanner redacts secret values in output by default
- SARIF reports are generated for integration with GitHub Security tab
